# tenantmanagementsystem
This was made by me as a final year project for BCA. This project was made using Node.Js-Express, MongoDB, EJS, HTML, CSS, JWT, etc. I intend make an advanced version of it with realtime notifications and possibly an SMS gateway, using React.
